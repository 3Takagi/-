# 密室毒杀案 Web 小游戏

基于 AI 生成的侦探推理网页游戏。

## 入口

打开 `index.html` 即可运行。

## 当前玩法

- 当前剧情根据 `案件文本_详细版.docx` 重写。
- 主线包含“星核系统”代码失窃、商业间谍身份、茶水间纸杯、匿名转账和三人交叉证词。
- 游戏以开场封面进入，主界面只显示当前章节场景、叙事文本和行动选项。
- 玩家按时间线逐步推进剧情，并可回到已解锁章节。
- 案卷、证据、人物、结案以弹窗形式打开。
- 结案使用选择式推理，判断主谋、协助者、知情隐瞒者、隐藏动机、下毒载体、密室形成方式和最终证据链。

## 正确结论

林薇是主谋和核心凶手；苏晴是协助者；王浩是知情隐瞒者。林薇因商业间谍身份暴露并需要窃取“星核系统”核心代码，策划杀害陈峰。苏晴因孩子医药费和挪用公款把柄协助递送毒纸杯、望风并收取报酬。王浩撞见陈峰濒死后因自身资金问题选择隐瞒。
